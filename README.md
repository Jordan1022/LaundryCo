# LaundryCo

Simple marketing site for Laundry Co built with Next.js 14 (App Router).

## Getting started

1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Visit `http://localhost:3000`

## Project structure

- `src/app/page.tsx` - Landing page sections
- `src/app/components/ContactSection.tsx` - Contact CTA + modal form
- `src/app/styles/ContactSection.module.css` - Styles for the contact components
- `src/app/page.module.css` - Layout and section styles

The contact form currently simulates a request so you can showcase the UI without needing a backend. Update the placeholder contact details before launch.
